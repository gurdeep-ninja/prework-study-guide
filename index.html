Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Element attribute class and id are similar
  • Most HTML elements are using opening and closing tags.
  • HTML elements can have attributes i.e. class, src etc
  • You can nest elements
  • You use <!-- to add HTML comments -->

CSS

  • There are 3 ways to add css to a web page: Inline, internal & external stylesheets
  • Inline and external sheets are referenced in the <head> section
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Learn CSS Layout

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stages all changes
  • git revert [SHA]: revert back to a previous commit SHA/hash
  • git clone [repository]: Clones a repository
  • git config --global user.name "Firstname Secondname": Sets the Git username
  • git config --global user.email "email@example.com": Sets the Git email
  • Git command cheat sheet
  • Branch naming convention: [type]/[name] i.e feature/starter-code or bug/broken-link

JavaScript

  • The 0 and null values are both falsy, so the computer won't execute the code between the proceeding curly brackets.
  • var.length is an array property representing the number of elements in the array.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.

Bash

  • ssh-keygen -t ed25519 -C "email@example.com": generate SSH key using Ed25519 encryption algorithm
  • eval "$(ssh-agent -s)": starts the ssh agent
  • ssh-add: adds ssh key to ssh agent
  • clip < file: copies file contents to clipboard

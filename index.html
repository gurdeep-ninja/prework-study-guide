Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Element attribute class and id are similar
  • Most HTML elements are using opening and closing tags.
  • HTML elements can have attributes i.e. class, src etc
  • You can nest elements
  • You use <!-- to add HTML comments -->

CSS

  • There are 3 ways to add css to a web page: Inline, internal & external sylesheets
  • Inline and external sheets are referenced in the <head> section
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stages all changes

JavaScript

  • The 0 and null values are both falsy, so the computer won't execute the code between the proceeding curly brackets.
  • var.length is an array property representing the number of elements in the array.
